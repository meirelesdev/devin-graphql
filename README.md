# Devin Graphql

> Servidor GraphQL construido com Apollo Server e Node.js, demonstrando criacao de schemas, queries, mutations e resolvers.

## Stack

- Apollo Server
- GraphQL

## Prerequisitos

- Node.js v14+

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/devin-graphql.git
cd devin-graphql

# Instale as dependencias
npm install

# Inicie a aplicacao
npm start
```
